# He Zhao Personal Academic Website

This is the source code for He Zhao's personal academic website, designed for GitHub Pages.

## Local files

- `index.html`: main website page
- `style.css`: website styling
- `assets/cv/ZHAO_He_CV.pdf`: downloadable CV file
- `assets/img/`: images, such as a profile photo

## Deployment

Create a GitHub repository named `flaneur.github.io`, then push this folder to the `main` branch.

The website will be available at:

https://flaneur.github.io
